# Characters

The runtime catalog contains only:

- Kris
- Lancer
- Ralsei
- Noelle
- Susie
- Spamton
- Jevil
- Tenna

Each character includes distinct GIFs for idle, walk, fast walk, run, swipe and
ball interaction. The default size is medium. Tenna receives a 2x relative
scale at every configured size.

Animation files live in `media/<character>/` and are built on transparent
220x220 canvases at approximately 12 FPS.
