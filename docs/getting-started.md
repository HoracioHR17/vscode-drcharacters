# Getting started

## Local development

1. Open the repository folder in VS Code.
2. Run `npm install`.
3. Press `F5` and choose **Run Extension**.
4. In the new Extension Development Host, open the command palette and run
   **Deltarune Characters: Start Deltarune character session**.

The extension starts with Kris, Dark World assets and a medium display size.
Choose another character through `vscode-pets.petType` or the spawn-character
command.
